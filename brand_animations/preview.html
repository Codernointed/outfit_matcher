Using these in Flutter

These are animated SVGs (CSS + SMIL). To get true Lottie-grade performance on mobile, you have three paths:

Easiest → Drop them in via flutter_svg for static + a wrapping AnimatedBuilder for motion. Works, but the SVG's internal CSS animations won't run — you'd re-implement the motion in Flutter.

Best for production → Rebuild the chosen 1-2 animations in Rive (free, Flutter-native, GPU-accelerated). Use the SVGs as design reference. Rive files load with the rive package.

If your team uses Lottie → Open the SVG in After Effects (via Overlord or SVG import), animate, then export to JSON via Bodymovin. Use the lottie package in Flutter.

For the splash screen specifically: render a static PNG of the logo as the native launch image (so it appears instantly during boot), then play the animated version once Flutter starts.
